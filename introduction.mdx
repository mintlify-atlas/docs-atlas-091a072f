## How It Works

vLLora acts as an intelligent proxy between your AI applications and LLM providers. It intercepts API calls, collects telemetry data, and provides real-time insights through a beautiful web interface.

<Steps>
  <Step title="Install vLLora">
    Install vLLora using Homebrew on macOS or Linux
  </Step>
  
  <Step title="Start the Server">
    Run `vllora` to start both the API server (port 9090) and the web UI (port 9091)
  </Step>
  
  <Step title="Configure API Keys">
    Visit the web UI to configure your AI provider API keys
  </Step>
  
  <Step title="Make Requests">
    Point your applications to `http://localhost:9090` and watch the traces flow in
  </Step>
</Steps>

## Architecture

vLLora is built with Rust for performance and reliability:

- **Gateway**: High-performance API proxy with OpenAI-compatible endpoints
- **Core**: Chat completion execution, routing, and streaming
- **LLM Crate**: Unified client library for multiple providers
- **Telemetry**: Real-time event collection and broadcasting
- **Guardrails**: Safety and validation layer for LLM interactions
