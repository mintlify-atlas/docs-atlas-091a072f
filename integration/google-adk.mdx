vLLora integrates seamlessly with Google's AI Development Kit (ADK) and Gemini models through its OpenAI-compatible API. This allows you to trace, debug, and monitor your Gemini-powered applications in real-time.

## Quick Start

Use Gemini models through vLLora with the OpenAI SDK format:

```python
from openai import OpenAI

# Point to vLLora gateway with Gemini model
client = OpenAI(
    api_key="your-gemini-api-key",  # Configure in vLLora UI
    base_url="http://localhost:9090/v1"
)

response = client.chat.completions.create(
    model="gemini-2.0-flash-exp",
    messages=[
        {"role": "user", "content": "Hello, Gemini!"}
    ]
)

print(response.choices[0].message.content)
```

## Configuration

### Setup vLLora

1. **Start vLLora**:
   ```bash
   vllora
   ```

2. **Configure Google API Key**:
   - Visit `http://localhost:9091`
   - Navigate to Settings/Providers
   - Add your Google AI/Gemini API key

## Streaming Responses

vLLora supports streaming with Gemini models:

```python
from openai import OpenAI

client = OpenAI(
    api_key="your-gemini-api-key",
    base_url="http://localhost:9090/v1"
)

stream = client.chat.completions.create(
    model="gemini-2.0-flash-exp",
    messages=[{"role": "user", "content": "Write a poem about AI"}],
    stream=True
)

for chunk in stream:
    if chunk.choices[0].delta.content:
        print(chunk.choices[0].delta.content, end="", flush=True)
```

## Function Calling

vLLora translates OpenAI-style function calls to Gemini's function calling format:

```python
from openai import OpenAI
import json

client = OpenAI(
    api_key="your-gemini-api-key",
    base_url="http://localhost:9090/v1"
)

tools = [
    {
        "type": "function",
        "function": {
            "name": "get_current_weather",
            "description": "Get the current weather in a given location",
            "parameters": {
                "type": "object",
                "properties": {
                    "location": {
                        "type": "string",
                        "description": "The city and state, e.g. San Francisco, CA"
                    },
                    "unit": {
                        "type": "string",
                        "enum": ["celsius", "fahrenheit"]
                    }
                },
                "required": ["location"]
            }
        }
    }
]

response = client.chat.completions.create(
    model="gemini-2.0-flash-exp",
    messages=[{"role": "user", "content": "What's the weather in Tokyo?"}],
    tools=tools
)

if response.choices[0].message.tool_calls:
    for tool_call in response.choices[0].message.tool_calls:
        print(f"Function: {tool_call.function.name}")
        print(f"Arguments: {tool_call.function.arguments}")
```

### Base64 Image Support

```python
import base64
from openai import OpenAI

def encode_image(image_path):
    with open(image_path, "rb") as image_file:
        return base64.b64encode(image_file.read()).decode('utf-8')

client = OpenAI(
    api_key="your-gemini-api-key",
    base_url="http://localhost:9090/v1"
)

base64_image = encode_image("path/to/image.jpg")

response = client.chat.completions.create(
    model="gemini-2.0-flash-exp",
    messages=[
        {
            "role": "user",
            "content": [
                {"type": "text", "text": "Describe this image in detail"},
                {
                    "type": "image_url",
                    "image_url": {
                        "url": f"data:image/jpeg;base64,{base64_image}"
                    }
                }
            ]
        }
    ]
)

print(response.choices[0].message.content)
```

## Multi-turn Conversations

```python
from openai import OpenAI

client = OpenAI(
    api_key="your-gemini-api-key",
    base_url="http://localhost:9090/v1"
)

conversation = [
    {"role": "system", "content": "You are a helpful math tutor."},
    {"role": "user", "content": "What is calculus?"}
]

# First exchange
response1 = client.chat.completions.create(
    model="gemini-2.0-flash-exp",
    messages=conversation
)

conversation.append({
    "role": "assistant",
    "content": response1.choices[0].message.content
})

# Follow-up question
conversation.append({
    "role": "user",
    "content": "Can you give me an example?"
})

response2 = client.chat.completions.create(
    model="gemini-2.0-flash-exp",
    messages=conversation
)

print(response2.choices[0].message.content)
```

## Async Support

Use async clients for concurrent requests:

```python
import asyncio
from openai import AsyncOpenAI

async def ask_gemini(question: str) -> str:
    client = AsyncOpenAI(
        api_key="your-gemini-api-key",
        base_url="http://localhost:9090/v1"
    )
    
    response = await client.chat.completions.create(
        model="gemini-2.0-flash-exp",
        messages=[{"role": "user", "content": question}]
    )
    
    return response.choices[0].message.content

async def main():
    questions = [
        "Explain quantum computing",
        "What is machine learning?",
        "How does blockchain work?"
    ]
    
    tasks = [ask_gemini(q) for q in questions]
    answers = await asyncio.gather(*tasks)
    
    for q, a in zip(questions, answers):
        print(f"Q: {q}")
        print(f"A: {a}\n")

asyncio.run(main())
```

## Using with LangChain

Integrate Gemini through vLLora with LangChain:

```python
from langchain_openai import ChatOpenAI
from langchain_core.messages import HumanMessage, SystemMessage

llm = ChatOpenAI(
    model="gemini-2.0-flash-exp",
    openai_api_key="your-gemini-api-key",
    base_url="http://localhost:9090/v1"
)

messages = [
    SystemMessage(content="You are a helpful AI assistant."),
    HumanMessage(content="Explain the theory of relativity simply.")
]

response = llm.invoke(messages)
print(response.content)
```

## Direct HTTP Requests

Alternatively, use direct HTTP requests:

```python
import requests
import json

url = "http://localhost:9090/v1/chat/completions"

payload = {
    "model": "gemini-2.0-flash-exp",
    "messages": [
        {"role": "user", "content": "What is AI?"}
    ],
    "temperature": 0.7
}

headers = {"Content-Type": "application/json"}

response = requests.post(url, json=payload, headers=headers)
data = response.json()

print(data["choices"][0]["message"]["content"])
```

## Troubleshooting

### API Key Configuration

Ensure your Google AI/Gemini API key is configured in vLLora UI at `http://localhost:9091`.

### Model Availability

Verify the Gemini model name is correct. Check [Google AI Studio](https://aistudio.google.com/) for available models.

### Connection Issues

Test vLLora connectivity:

```bash
curl http://localhost:9090/health
```

### Rate Limits

vLLora respects Google AI API rate limits. Monitor usage in the vLLora UI and implement exponential backoff for production applications.

### Image Upload Errors

For large images, ensure they're properly encoded and under Google's size limits. Consider resizing images before sending.
